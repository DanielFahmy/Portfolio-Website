# Daniel Fahmy – Portfolio Website

## 📖 Overview
A personal portfolio website built to showcase my projects, skills, and background as a Computer Science student and aspiring frontend/ full-stack software developer.  
The site is fully responsive, easy to navigate, and designed with a clean, minimal layout to highlight my work.

---

## 🛠 Tech Stack
- HTML5  
- CSS3  
- JavaScript (ES6)  

---

## ✨ Features
- Responsive layout for desktop and mobile devices  
- Interactive navigation bar with clickable sections  
- Smooth scrolling between sections  
- Clean and minimal UI design  
- Project showcase with links to GitHub repositories and live demos  
- Contact section with direct email and social links  

---

## 🧭 Website Sections
- **Home** – Introduction and personal headline  
- **About** – Brief background and interests  
- **Skills** – Front-end, back-end, and database technologies  
- **Portfolio** – Featured projects with descriptions and links  
- **Contact** – Email, LinkedIn, GitHub, and resume download  

---

## 🔧 What I Worked On
- Designed and structured the website using semantic HTML  
- Styled the layout and components using custom CSS  
- Implemented interactive behavior and navigation using JavaScript  
- Organized content into clear, user-friendly sections  
- Deployed the site usibng Netlify for public access
